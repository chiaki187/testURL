これはeffectアニメーションとspacialアニメーションを組み合わせた場合のテストモーダルを実装したものである。
effectアニメーションとはコンポーネントの移動ではなく、透明度や色の変化のアニメーションを示す。
spacailアニメーションとは移動や回転などの動きを差す。

このプログラムではモーダルの動きをspacilアニメーション、モーダルとともに画面の暗さの変化をeffectアニメーションを差しており、それぞれ異なったイージングの実装を行っている。
イージングの動きについてはMaterial3を参考にした。

結果としては、イージングの組み合わせを変えても、印象に大きな違いは生じなかった。
